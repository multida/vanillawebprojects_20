## 영화 좌석 예매

티켓을 구입하기 위해 영화 선택 및 극장 좌석 표시

## 프로젝트 작업

- [ ] 영화 선택, 화면, 좌석, 범례, 좌석 정보가 포함된 UI 작성
- [ ] 사용자는 영화/가격을 선택할 수 있습니다.
- [ ] 사용자는 좌석을 선택/취소할 수 있습니다.
- [ ] 사용자는 점유석을 선택할 수 없습니다.
- [ ] 좌석 수와 가격이 업데이트됩니다.
- [ ] 좌석, 영화, 가격을 로컬 저장소에 저장하여 업데이트 시 UI가 계속 로드되도록 합니다.
